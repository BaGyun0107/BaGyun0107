### 💡 I Focus On...
- **Architecture & Reliability:** 단순한 API 개발을 넘어 결제 트랜잭션, 간이 Outbox 패턴, Optimistic Lock을 활용한 동시성 제어 등 데이터 정합성 보장에 집중합니다.
- **Problem Solving:** 표면적인 에러가 아닌 네트워크/인프라 계층(HTTPS, Cloudflare 등)의 근본 원인을 파악하고 해결합니다.
- **AI-Driven Development:** 바이브 코딩(Vibe Coding)과 AI 에이전트를 오케스트레이션하여 생산성을 극대화하고, 철저한 TDD와 클린 아키텍처로 품질을 통제합니다.
